Final Project

This project showcases a colorful collection of images that represent different shades from the visible spectrum. It demonstrates how a responsive website can adjust its layout and style based on screen size and user preferences like reduced motion and dark mode.

The site uses modern CSS techniques such as grid layout, media queries, and accessibility features. It is designed to be mobile-first and adapts smoothly across mobile, tablet, and desktop views while supporting accessibility needs.
